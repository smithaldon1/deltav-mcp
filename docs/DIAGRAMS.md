# Diagrams

The server can generate Mermaid text for:

- module relationships
- cause-and-effect diagrams
- sequence logic
- mode/state transitions
- batch phase flow
- alarm response flow

Tools:

- `generate_module_relationship_diagram`
- `generate_cause_effect_diagram`
- `generate_sequence_logic_diagram`
- `generate_mode_state_diagram`
- `generate_batch_phase_diagram`
- `generate_alarm_response_diagram`

Output is Mermaid text only. No image rendering or external rendering services are used.
